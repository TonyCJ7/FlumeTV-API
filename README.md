# flume-tv-frontend
A feature-rich, self-hostable Stremio IPTV backend that ingests M3U playlists and optional EPG (XMLTV) guide data — with Xtream Codes support, encrypted UUID-only addon tokens, SQLite-backed catalog sync, background prefetch workers, scheduler/queue, Server-Sent Events for sync progress &amp; logs, and a REST API for the FlumeTV config UI.
